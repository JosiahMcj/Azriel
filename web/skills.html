Skills are named workflows: a kickoff prompt + persona + style + tool hints, packaged so you can launch a focused session in one click. Launching a skill creates a fresh chat session pre-seeded with the kickoff prompt; you tweak it, hit send, and Azriel takes it from there.

Walks you through it: tell Azriel the workflow you want, iterate on name + kickoff + voice, hit done. The new skill saves into "Your skills" below automatically.
